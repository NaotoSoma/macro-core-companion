# Lecture Page Template

各講義ページは、原則として以下の順序で作る。

## 1. Frontmatter

必須または推奨:

- title
- description
- week
- section
- estimatedTime
- learningGoals
- keywords
- datasets
- widgets
- status

## 2. Body structure

### 今日の問い

学生が 1 行で答えられる問いにする。

### 直感

数式より先に、何が問題なのかを短く説明する。

### 最小モデル

2期間または 3期間の設定を中心に置く。

### interactive widget

1つの概念だけを動かす。複数の新概念を混ぜない。

### データで確かめる

時系列 1 枚、散布図または分布図 1 枚を目安にする。

### 演習

確認問題を 2〜4 個置く。

### 一般化

無限期間版や研究フロンティアへの橋渡しを 1 段落で示す。

## 3. Writing rules

- 段落は短くする。
- 箇条書きは必要最小限。
- 英語の専門用語には日本語を添える。
- 1ページの中核メッセージは 1 つに絞る。
