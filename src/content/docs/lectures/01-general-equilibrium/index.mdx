# 直感

経済を理解する第一歩として、まず家計や企業の意思決定をそれぞれ個別に考えることは有用です。ここでは、賃金、利子率、物価などをひとまず所与として、各主体がどのような需要や供給を生み出すかを考えます。

そのうえで、ある1つの市場だけを取り出し、他の市場の状況を固定したまま需要と供給の交点を考えるのが**部分均衡（partial equilibrium）**です。

しかし、マクロ経済学では、**ある市場での変化が他の市場を通じてさらに経済全体に波及し、最後には元の主体の選択条件そのものを変えてしまう**ことが重要です。たとえば、家計が将来不安から消費を減らして貯蓄を増やすと、資産市場で資金供給が増えて利子率が下がるかもしれません。すると企業の投資や労働需要が変わり、雇用や賃金を通じて家計所得も変わります。その結果、財市場での消費需要や家計の貯蓄行動も再び変わりえます。

このように、**1つの市場で起きた変化が他の市場を通じて跳ね返ってくる**ときには、1つの市場だけを見る部分均衡では十分ではありません。そこで、複数の市場と複数の主体の行動を同時に整合的に考える**一般均衡（general equilibrium）**の見方が必要になります。

この回の最小モデルは、家計・企業・政府・中央銀行をそれぞれ一つずつの代表的主体（representative agent）として置いた、**経済全体の見取り図**です。重要なのは細かい数式ではなく、同じ価格体系のもとで4者の行動が両立するかを追うことです。

## 各経済主体の意思決定問題

### 1. 家計

賃金や利子や配当から決まる所得制約のもとで、効用を最大化するように

- 労働を供給する
- 消費と貯蓄を選ぶ

### 2. 企業

技術制約のもとで、利潤を最大化するように

- 労働や資本を使って生産する
- 賃金を支払う
- 生産物を販売する

### 3. 政府

予算制約を守るように

- 支出を行う
- 税を徴収する
- 必要なら国債を発行する


この三つの経済主体を、それぞれ独立に考えるだけでは足りません。重要なのは、**同じ価格体系のもとで、すべての経済主体の行動が同時に成立するかどうか**です。

## 同時に確認する市場

この見取り図を閉じるために、少なくとも3つの市場を同時に見ます。

1. 財市場: 企業が生産した財の量が家計が需要する財の量と一致しているか
2. 労働市場: 家計の労働供給と企業の労働需要が一致しているか
3. 資産市場: 貯蓄・投資・債券の需給が一致しているか

1つだけを見れば部分均衡（partial equilibrium）ですが、複数市場を同時に整合させようとすると、一般均衡（general equilibrium）の視点に近づきます。

# データで確かめる

この回では、まだ厳密な実証までは進みません。その代わり、今後の授業で次のような系列を並べて見ることを予告します。

1. 実質GDP、消費、投資の時系列
2. インフレ率、失業率、政策金利の時系列
3. 実質賃金と労働生産性の動き

ここでのポイントは、これらが**別々の系列ではなく、相互に結びついた系列**だということです。今後の各回で、小さいモデルを使いながら、その結びつきを少しずつ明示していきます。

# 演習

1. 部分均衡（partial equilibrium）と一般均衡（general equilibrium）の違いを、数式を使わずに説明してください。
2. 企業の雇用削減が、労働市場以外にも影響すると考えられる理由を説明してください。
3. 政府支出を増やしたときに、財市場以外で何を同時に確認する必要があるかを挙げてください。

# 一般化への橋渡し

次回からは、この見取り図の中の**家計**に焦点を当てます。まずは 2期間モデルを使い、消費、労働、貯蓄の選択がどのように決まるかを考えます。

そこで初めて、異時点のトレードオフと、マクロモデルで繰り返し出てくる最適化の骨格が登場します。
